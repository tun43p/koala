Usage :           koala                   Run Koala Toolkit
                  koala [argument]        Options

Arguments :
  -b :            Build
  -e :            Edit Dockerfile
  -h :            Print help
  -k :            Kill containers
  -r :            Remove containers
  -R :            Remove containers and image
  -u :            Upgrade Koala
  -uninstall :    Uninstall Koala
  -v :            Print current version
